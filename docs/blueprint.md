# **App Name**: HealthWise Companion

## Core Features:

- AI Symptom Checker: Allows users to chat about their health and receive educational, non-diagnostic symptom guidance using a generative AI tool.
- Daily Health Notification: Provides daily reminders and motivational messages related to health, such as hydration and physical activity. The notification time should be customizable by the user.
- Activity Tracker: Tracks user's physical activities like walking, running, and cycling, including distance and duration.
- Weather Alert: Fetches weather information and provides contextually relevant health advice, such as staying hydrated in hot weather.
- User Profile: Displays user information, a summary of recent AI chats and activities, and controls for enabling/disabling notifications.
- User Authentication: Enables user login/signup using email/password and Google Sign-In.
- Data Persistence: Stores user data (profile, activities, chats) in Firestore.

## Style Guidelines:

- Primary color: Deep sky blue (#329DFF) for trustworthiness and calmness.
- Background color: Very light blue (#E5F3FF) for a clean and calming backdrop.
- Accent color: Light green (#90EE90) for highlighting positive health messages and calls to action.
- Body and headline font: 'PT Sans' for a modern, yet warm and accessible feel, suitable for both headlines and body text.
- Use Material Design Icons (MDI) for a consistent and recognizable visual language related to health, chat, and weather.
- Employ rounded corners and soft shadows for a modern, friendly look, in line with Material 3 design principles.
- Use subtle animations for UI transitions to enhance the user experience, such as chat bubbles and activity updates.
